Running Man

==============================================================================
You act as a frog and move the frog by press left, right, up and down key in the keyboard. You can press CONTROL to shoot missiles, which to eliminate the arrows that hit by missiles. You can also press B to activate bomb that used to clean all the arrows and mushrooms in the screen. But you only get 2 bombs at a time. Your goal is trying to make the frog move to the right end of the screen.
==============================================================================

Install instruction:

1. You need Java8 or JDK1.8 in your machine.
2. Compile all the java files in the src folder
3. Run the main function of RunningManEx.
4. You are ready to go!
